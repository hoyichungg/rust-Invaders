# rust-Invader

## 簡介
rust-Invader 是一款用 Rust 語言編寫的簡單街機遊戲。靈感來自於經典的太空侵略者遊戲，玩家需要控制一艘太空船來摧毀一波又一波的外星人。

## 特色
- 簡單直觀的遊戲玩法。
- 復古圖形和音效。

## 系統要求
要運行 Rust-Invader，您需要在您的電腦上安裝 Rust。您可以通過官方網站安裝 Rust：[安裝 Rust](https://www.rust-lang.org/tools/install)。

## 安裝

```bash
git clone
cd rust-invader
cargo run
```
